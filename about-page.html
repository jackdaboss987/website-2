Why I am making a Cheat Sheet

Last year when doing Digital Technology I found it very hard to find exemplars based the current projects I was doing at the time. So to save the new students that are taking Digital Technology I am creating this website to post specific code based on my previous projects to help the students improve their overall quality of the code.

In the near future there will be an option for students to upload their own code with a brief explaination of what the code does so that all of the previously added code doesn't become outdated and irrelevant as the Standards change and progress.

What the Cheat Sheet will outline

This sheet will include help based on three main coding languages: Python, Html and CSS as these are the language most used in the projects.

As it has been a while since I have coded in Python the content and help I can provide will be very limited.

Html is very basic so the content will be short and sharp making it easy to find what you are looking for.

CSS can be very difficult at times so I will slowly start to add more content to the page to provide extra help.
