HTML ID's and Classes

Adding a ID to an HTML element instead of just using div makes it much easy to find and style when using CSS.

Here is some examples:

            <header class="title">
            <h1 id="box-title">

Setting up a basic website

There are many ways to setup a website but I will show you the quickest. First create a CSS and HTML and link the two using the first code example. Then you can give your website a title by following the second example.

        <link rel="stylesheet" href="example.css">
        <title>Example</title>
